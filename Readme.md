**Project 1:** Advanced EDA & Feature Engineering
**Objective**

The objective of this project is to transform raw data into a clean and machine-learning-ready dataset through exploratory data analysis and feature engineering.

**Tasks Performed**
Data loading 
Missing value handling
Outlier detection and treatment using IQR
Feature engineering
Data visualization
Dataset cleaning and export

**Feature Engineering**

The following new features were created:

HasCoupon
OrderMonth
OrderDayOfWeek

Python
Pandas
NumPy
Matplotlib
Seaborn
